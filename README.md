# ApoB
This program calcualates your ApoB score (mg/dL) using:
```
1. Total chloesterol
2. HDL cholesteral
3. Tryglicerides
```

## Usage

The flags used in the CLI are:
```
    -tc :   Total Cholesterol.
    -hdlc : High-Density Lipoprotein Cholesterol.
    -tg :   Triglycerides. 
```
Commands to run code:
```Code
% cd your_uv_project/apob
% chmod +x apob.py 
% uv run pytest -sv
% uv run apob.py
% uv run apob.py -tc 155 -hdlc 30 -tg 135
```

## ApoB Calculation
Generated by Google AI:

ApoB=(0.65 x TC) - (0.59 x HDLC) + (0.01 x TG)

The provided formula is a regression equation used to estimate the plasma concentration of
Apolipoprotein B (ApoB) in milligrams per deciliter (mg/dL). This calculation is typically employed in clinical settings when direct laboratory measurement of ApoB is not available. 

➡️ Step 1: Variable Identification
To use this formula, obtain the following values from a standard lipid panel, ensuring the units are in mg/dL: 

    TC :   Total Cholesterol.
    HDLC : High-Density Lipoprotein Cholesterol.
    TG :   Triglycerides. 

➡️ Step 2: Application of Coefficients
Multiply each lipid parameter by its respective weighting factor derived from clinical studies:

    1. Total Cholesterol contribution: Multiply TC by 0.65.
    
    2 HDL Cholesterol offset: Multiply HDLC by 0.59.
    
    3. Triglyceride contribution: MultiplyTG by 0.01.
    

➡️ Step 3: Final Computation
Combine the values to determine the estimated concentration:
    ApoB=(0.65 x TC) - (0.59 x HDLC) + (0.01 x TG)

✅ Answer:
The formula ApoB=(0.65 x TC) - (0.59 x HDLC) + (0.01 x TG) provides an estimate of the total number of atherogenic lipoproteins (including LDL, VLDL, and IDL) in the bloodstream. Because each atherogenic particle contains exactly one molecule of Apolipoprotein B, this value is often considered a more accurate predictor of cardiovascular risk than LDL-C alone, especially in patients with high triglycerides or diabetes. 


ApoB (Apolipoprotein B) levels measure the number of atherogenic particles in the blood, often serving as a better indicator of cardiovascular risk than LDL cholesterol alone. Optimal levels are generally < 90 - 100 mg/dL, with high risk indicated at >= 130 mg/dL. For individuals with high cardiovascular risk or established disease, experts often recommend aiming for < 80 mg/dL or lower.

**ApoB Reference Ranges (mg/dL)**

    * Optimal/Low Risk: < 90 mg/dL (some sources suggest < 80 mg/dL>).
    * Borderline/Moderate Risk: 90 - 129 mg/dL.
    * High Risk: >= 130 mg/dL.
    * Very High Risk: > 150 mg/dL. 

**Contextualizing Your Numbers**
    Target for High Risk: Patients with diabetes or established cardiovascular disease (CVD) should aim for an ApoB < 80 mg/dL.
    The "Lower is Better" Approach: Experts like Dr. Peter Attia and some cardiologists argue for getting ApoB as low as possible for long-term prevention, often citing levels around 60 mg/dL or lower for optimal health.
    Gender/Age Factors: Typical ranges for adults are roughly 60 - 117 mg/dL for women and 66 - 133 mg/dL for men. 

**Why ApoB Matters**
Unlike standard LDL tests that measure cholesterol content, ApoB directly measures the number of particles (VLDL, IDL, LDL) that cause plaque buildup. A person can have a "normal" LDL-C number but a high ApoB, indicating a high concentration of small, dense particles.
